Configure Test Stand

The Test Stand configuration is splitted in 2 differents screens. When opening these screens you are connected to the Test Stand and retrieving the Test Stand configuration stored on the board memory

Test Stand config

- Baud Rate Allow to change the Test Stand communication speed (reboot required)

- Test Stand Res The Test Stand resolution (number of measures per seconds

- EEprom size Allow you to change the on board memory size

- Recording duration Recording time in second

- Battery type Define the battery type used, the is used by some Test Stand to warn on battery low voltage
