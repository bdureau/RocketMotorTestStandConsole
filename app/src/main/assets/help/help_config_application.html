Configure Bear Console Application

The configuration is divided in 2 screens

General

- Application langage (not used for now because the application langage will be the phone language)

- Graph color

- Application Units

- Graph background color (make sure that it is different from the graph color!!!)

- font size

- Baud rate (this is for the USB cable, it has to be the same as the altimeter baud rate)

- Connection type can be bluetooth or USB if you want to do long distance telemetry)

- Graphics library (allow to choose between 2 lib type, be carefull that on Android 8 and above you will crash your phone if you do not use the MPAndroid chart lib)

- Multiple Main/drogue Alow you to define more than one output as a main or a drogue

- Full USB support Only enable it if you use USB with a cable. If using longue range telemetry do not use it to avoid corrupting your altimeter configs

Telemetry config

Allow to configure which telemetry event are said. Allow also to change the telemetry voice. If you want to test the telemetry voice just press the [Test voice] button

Whenever you are done click on the save button. You can also exit the config without saving or restore it to the default one.
