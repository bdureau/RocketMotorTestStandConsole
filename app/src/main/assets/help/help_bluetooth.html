Select bluetooth module

You will be prompted to choose a bluetooth module if you have not selected one already or if you want to connect to a different Test Stand.

You first need to pair all you bluetooth Test Stand modules in your Android device bluetooth configuration. Make sure that you can clearly differenciate them so that your connect to the correct Test Stand

From the list of configured modules select the one your Test Stand is using, leave the screen and connect again. If you have choosen the wrong module it will not connect.
