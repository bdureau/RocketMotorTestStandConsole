Configuration du banc de test

La configuration du banc de test est répartie sur 2 écrans. Lorsque vous ouvrez chaque écran vous êtes connecté au banc de test et consultez la configuration stockée dans la mémoire du banc de test

Configuration du banc de test

- Baud Rate Permet de changer la vitesse de communication du banc de test(reboot nécessaire)

- Res banc de test La résolution du banc de test

- Capacité EEprom Permet de changer la taille de l'EEprom sur la carte

- Durée d'enregistrement Permet de définir la durée max d'enregistrement

- Type de batterie Cela permet pour certain banc de test de prévenir lorsque la tension de la batterie chute

Configuration du banc de test (page 2)

Donne des informations concernant la version du banc de test

Permet de choisir les unités du banc de test
