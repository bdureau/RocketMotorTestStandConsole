Configure altimeter

The altimeter configuration can be up to 4 differents screens depending of the altimeter you are connected to. When opening these screen you are connected to the altimeter and retrieving the altemiter configuration stored on the board memory

Altimeter Outputs

Each output of the altimeter can be configured and assigned a functionnality. Fonctionnalities are event driven. The following events are available:

- Lift off (fire at lift off)

- Timer (fire X miliseconds after liftoff)

- Altitude (fire at the specified altidude)

- Apogee (fire at apogee)

- Main (fire X meters before landing)

- Landing (fire on landing)

Note that for each events you can also have a delay in miliseconds

Altimeter config

- Baud Rate Allow to change the altimeter communication speed (reboot required)

-Apogee measures The number of measures required to detect apogee

- Altimeter Res The altimeter resolution(be carefull changing it will reduce the number of measures do not do it if you use the altimeter to deploy parachutes)

- EEprom size Allow you to change the on board memory size

- Beep ON/Off You can turn ON or OFF the continuity beep

- End of rec altitude At which altitude do you stop recording

- Record temp

- Supersonic delay

- Lift Off Altitude At which altitude do we consider that lift off as occured

- Battery type Define the battery type used, the is used by some altimeters to warn on battery low voltage

Altimeter config (page 2)

Allow to choose the altimeter units, the beep mode and the beep frequency

Servo config
