Fichier Aide de la Console Bear Altimeter

Ceci est une aide rapide pour utilser votre console Bear altimeter sur Android. La console Android est très pratique à utiliser pour reconfigurer votre altimètre lorsque vous êtes sur le terrain.

Tout d'abord vous devez vous assurer que votre altimètre est compatible (c'est à dire que le bon firmware a été chargé).

Ensuite assurez vous que le module bluetooth est connecté à l'altimètre. 

Puis vous devez aller dans la configuration de votre appareil Android et dans la configuration bluetooth chercher les module bluetooth HC-05, HC-06 ou HC-07.

Lorsque vous l'avez trouvé entrez le code pour associer le module. Le code par dèfaut est 1234.

!!! ATTENTION !!!!

Il est souhaitable de nommer clairement chaque adaptateur bluetooth sur votre téléphone de manière à vous assurer que vous configurez le bon altimètre. 

Ensuite allumer l'altimètre et attendre jusqu'à ce qu'il bip la continuité.

Puis démarrer l'application sur votre appareil Android, cliquer sur montrer les périphériques et choisissez le module HC-05 ou HC-06. L'application tente ensuite de se connecter à l'altimètre.

Si le message connecté apparait, cela veut dire que vous êtes connecté et que vous pouvez utiliser la Console pour configurer votre  altimètre ou lire vos vols. Si cela n'arrive pas à se connecter assurez vous que vous avez selectionné le bon module bluetooth ou que l'altimètre est sous tension.

Utilisation de la console altimètre

Ecran principal de l'altimètre

Sur l'écran principal vous avez un certain nombre de boutons, décrivons les. 

[Configuration Altimètre]

Charge la configuration de l'altimètre et vous permet de la modifier.

[Chargement Vols]

Charge la totalité des vols que vous avez enregistré et les affiche.

[Télémetrie]

Télémetrie de l'altimetre

[Continuité Marche/Arret]

Allume ou éteind la continuité.

[Réinit Config]

Donne la possibilité de restaurer les paramètres d'usine de la configuration de l'altimètre. Cela permet également d'effacer tous les vols. 

[Etat]

Donne l'etat de l'altimétre, permet de tester les sorties

[Flash Firmware]

Permet de mettre à jour le firmware à l'aide d'un cable USB. Pour le moment uniquement les altimètres à base de ATMega328

[Connexion/Déconnexion]

Se connecte ou déconnecte de l'altimètre

Menu de l'Ecran principal de l'altimètre

Depuis le menu vous pouvez accèder à ce fichier d'aide, un écran à propos qui vous donnera des informations concernant la console et sa version . Vous avez également la possibilité de changer l'apparance de l'application.

 

 

La console est compatible avec les altimètres suivant: - Altiduo (version avec un Atmega328) - AltiMulti V1 - AltiMulti V2 - AltiServo - AltiGPS - AltiMulti SMT32
