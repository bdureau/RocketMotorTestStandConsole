Fichier Aide de l'application pour votre banc de test moteur

Ceci est une aide rapide pour utilser votre application Test stand sur Android. La console Android est très pratique à utiliser pour reconfigurer votre banc de test lorsque vous êtes sur le terrain.

Tout d'abord vous devez vous assurer que votre banc de test est compatible (c'est à dire que le bon firmware a été chargé).

Ensuite assurez vous que le module bluetooth est connecté au banc de test. 

Puis vous devez aller dans la configuration de votre appareil Android et dans la configuration bluetooth chercher les module bluetooth HC-05, HC-06 ou HC-07.

Lorsque vous l'avez trouvé entrez le code pour associer le module. Le code par dèfaut est 1234.

!!! ATTENTION !!!!

Il est souhaitable de nommer clairement chaque adaptateur bluetooth sur votre téléphone de manière à vous assurer que vous configurez le bon banc de test. 

Ensuite allumer le banc de test et attendre jusqu'à ce qu'il arrete de bipper .

Puis démarrer l'application sur votre appareil Android, cliquer sur montrer les périphériques et choisissez le module HC-05 ou HC-06. L'application tente ensuite de se connecter au banc de test.

Si le message connecté apparait, cela veut dire que vous êtes connecté et que vous pouvez utiliser la Console pour configurer votre  banc de test ou lire vos courbes de poussées. Si vous n'arrivez pas à vous connecter assurez vous que vous avez selectionné le bon module bluetooth ou que le banc de test est sous tension.

Utilisation de l'application pour votre banc de test

Ecran principal du banc de test

Sur l'écran principal vous avez un certain nombre de boutons, décrivons les. 

[Config Banc de Test Moteur]

Charge la configuration du banc de test et vous permet de la modifier.

[Chargement Courbes de Poussées]

Charge la totalité des courbes de poussées que vous avez enregistré et les affiche.

[Télémetrie]

Télémétrie du banc de test

[Réinit Config]

Donne la possibilité de restaurer les paramètres d'usine de la configuration du banc de test. Cela permet également d'effacer toutes les courbes de poussées. 

[Etat]

Donne l'etat du banc de test

[Flash Firmware]

Permet de mettre à jour le firmware à l'aide d'un câble USB. Pour les bancs de test à base de ATMega328 ou STM32

[Connexion/Déconnexion]

Se connecte ou déconnecte du banc de test

Menu de l'Ecran principal du banc de test

Depuis le menu vous pouvez accèder à ce fichier d'aide, un écran à propos qui vous donnera des informations concernant la console et sa version . Vous avez également la possibilité de changer l'apparance de l'application.

 

 

La console est compatible avec les bancs de test suivant:

 

  - Carte avec un Atmega328

  - Carte avec un SMT32
