Test Stand help file

This is a quick help file to use your Test Stand application. The Test Stand application is a very convenient way to record your motor thrust curves when you are on the field.


First of all you need to make sure that your Test Stand board is compatible (ie: the correct firmware has been loaded).

Secondly make sure that the bluetooth module is connected on the Test Stand board.

Thirdly you need to go to your android device settings, go to the bluetooth configuration and search for the HC-05, HC-06 or other bluetooth module. When you have find it enter the pairing code which by default is 1234.

!!! CAUTION !!!!

You might want to clearly name each bluetooth device paired in your phone to make sure that you configure the correct teststand.

 

Then turn on the TestStand and wait until it stops beeping.

Then start the application on your Android device, click on the show paired devices and finally select the HC-05 or HC-06 device. It now attempt to connect to the altimeter.

If it says connected, you are now in and can use the application to configure your Test Stand or read the recorded thrust curves. If it cannot connect then make sure that you have the correct bluetooth device selected or that the teststand board is on.

 

Using the Test Stand application

Test Stand main screen

On the main screen you have a number of buttons, let's go and describe each one.

[Test Stand Setting]

Read the Test Stand setting and allow you to modify them.

 

[Read Thrust Curves]

Read all the thrust curves that have been recorded and plot them.

 

[Telemetry]

Go to the Test Stand telemetry.  

 

[Reset Settings]

Give the ability to restore to factory settings of the Test Stand config. It also allow to clear all Thrust Curves. 

 

[Status]

Go to the Test Stand state screen.  

 

[Flash Firmware]

Allow to flash the firmware using a USB cable. This is currently supported only for non SMT32 board

 

[Connect/Disconnect]

Disconnect from the Test Stand

 

Test Stand main screen menu

From the menu you can access this help file, an about screen that will give you informations about the application and its version and finally you have the option to customize the look and feel of this application.

 

 

Supported Test Stand board for this application are:

 

- Test Stand with an Atmega328

- Test Stand with an SMT32
