Flash firmware

First you need to select the firmware. The firmware depend on the Test Stand that you want to flash

Then decide if you want to directly flash your firmware or use a recovery firmware first

Use a recovery firmware if the config is wrong and the Test Stand can no longer connect.

After using a recover firmware you will need to flash the firmware.

Note that flashing the wrong firmware will result in a Test Stand not working at all or not working as expected

In order to be able to flash your Test Stand you will need a USB cable and an adapter to plug it to your phone.

The following Test Stand can be flashed, please make sure that your carefully look at the pictures to identify your Test Stand

ATMega328 Test Stand (Large version)

STM32 Test Stand V1

STM32 Test Stand V2

Alternatively you can attempt to detect the firmware.

At the time of writing, 2 types of Test Stand exists, the ATMega328 based Test Stand and the STM32 based Test Stand.

Flashing ATMega328 based Test Stand

- Plug the cable

- Select the correct firmware or detect it.

- Press and release the reset button on the Test Stand and press the [flash firmware] button

Flashing STM32 based Test Stand

- Plug the cable

- Select the correct firmware or detect it

- Move the boot0 jumper to the right

- Press the reset button

- Press the [flash firmware] button

- Move the boot0 jumper back to the left
