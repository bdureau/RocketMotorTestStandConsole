Paramétrage de l'application Test Stand

La configuration est sur 1 écran

Générale

- Langue de l'application(non utilisé pour le moment car la langue de l'application est la langue du téléphone)

- Couleur du graphique

- Unité de l'application

- Couleur de fond du graphique (attention à ce que cela ne soit pas la même que la couleur du graphique!!!)

- Taille de la police

- Baud rate (Ceci est pour le câble USB, cela doit être identique au baud rate du banc de test)

- Le type de connexion qui peut être bluetooth ou USB si vous voulez faire de la télémétrie longue distance ou lire les données avec un câble)

- Bibliothèque graphique (permet de choisir entre 2 bibliothèques graphique; attention pour les versions 8 et suivantes d'Android votre téléphone crashera si vous n'utilisez pas la bibliothèque MPAndroid chart)

- Support USB complet Activez le uniquement si vous utilisez un câble USB. Si vous utilisez un module de télémétrie longue distance ne l'activez pas pour éviter de corrompre la configuration de votre banc de test

Lorsque vous avez terminé cliquer sur le bouton sauvegarder. Vous pouvez également quitter la config sans sauvegarder ou réinitialiser la config par défaut.
